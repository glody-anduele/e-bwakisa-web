Comment ça marche avec e-Bwakisa

Portail Bienvenue

Ouverture de compte

Portail de bienvenue,
ici vous avez deux boutons d’utilisation : la première Identifiez-vous si vous avez déjà un compte e-bwakisa valide et que vous avez réinstallé ou changé de téléphone vous pouvez cliquer dessus pour insérer vos deux comptes USD et CDF à destination de la vérification.
Cependant, si vous n’avez pas de compte e-bwakisa, veuillez vous renseigner auprès de l’agent dans l’agence pour qu’il vous active.
Après avoirfourni toutes les informations, il est recommandé de demander l’ouverture de compte auprès de l’agent.

Connexion

Activation du compte

Pour vous connecter à votre compte e-Bwakisa, une fois l'activation de votre compte effectuée par un agent,
vous disposerez de deux informations d'identification :
1. Un identifiant (ID)
2. Un mot de passe.
C'est en utilisant ces deux informations que vous pourrez protéger votre compte. Si vous avez oublié votre identifiant ou mot de passe,
Cliquez en bas de la zone de saisie de mot de passe, puis cliquez sur mot de passe oublié pour récupérer votre compte.

Demande de recuperation de mon Compte

Formulaire de demande

Pour récupérer votre mot de passe, il vous faudra remplir le formulaire de demande de réinitialisation de mot de passe, ce formulaire vous enverra un code par SMS et par e-mail. Ce code vous permettra d'accéder à la page de modification de votre mot de passe. Une fois que vous aurez envoyé votre demande, la fenêtre s’ouvrira et vous insérerez le code que vous avez reçu.

Mes Soldes

Les transactions

Dans cette page, vous avez une vue de vos avoirs en dollars américains (USD) et en francs congolais Congolais (CDF). Chaque fois que vous effectuez des dépôts et des retraits, la vue de vos avoirs sera mise à jour.

Acceuil

Preuves des transactions

1. Mes relevés:
ils vous serviront de preuves de vos retraits ou dépôts effectués auprès d'un agent e-bwakisa Grace Creative.
2. Relevés de transfert:
ces relevés de transfert sont des preuves des transferts que vous avez réalisés pour les comptes de vos bénéficiaires
auprès des agents e-bwakisa Grace Creative.
3. Messagerie:
vos messages de confirmation de vos dépôts seront affichés là.
4. Retrait en CDF:
Retrait en francs congolais, ouvre une nouvelle page à partir de laquelle vous pourrez effectuer vos transactions
de retrait en francs congolais auprès d'un agent e-bwakisa Grace Creative.
5. Retrait en USD:
Retrait en dollars américain, ouvre une nouvelle page à partir de laquelle vous pourrez effectuer vos transactions
de retrait en dollars américains auprès d'un agent e-bwakisa Grace Creative.

Envoi d'argent auprès d'un proche

Les étapes à suivre pour envoyer l'argent

Pour envoyer de l'argent à un proche, procédez comme suit :
Sélectionnez le type de monnaie en fonction du compte du bénéficiaire et cliquez sur le bouton correspondant au nom de l'abréviation monétaire (CDF ou USD) en haut.
1. Insérez le numéro de compte du bénéficiaire.
2. Insérez le montant à envoyer.
3. Cliquez sur le bouton « Envoyer ».

Retrait d'argent auprès d'un agent

Les étapes à suivre pour retirer l'argent

Pour effectuer un retrait d'argent auprès d'un agent, suivez ces étapes simples :
1. Insérez le numéro de l'agent en fonction du type de monnaie que vous souhaitez utiliser.
2. Insérez le montant que vous souhaitez retirer.
3. Cliquez sur le bouton "Valider".

Transfert d'argent

Transfert d'argent dans un compte bénéficiaire

Pour transférer de l'argent vers le compte d'un bénéficiaire, procédez comme suit :
1. Insérez le numéro de l'agent concerné ou demandez à votre bénéficiaire de récupérer un numéro d'agent dans l'agence pour vous
2. Insérez votre nom complet
3. Insérez le nom complet du bénéficiaire
4. Insérez le montant à transférer selon le type de monnaie souhaité (CDF ou USD)
5. Insérez un code de récupération pour le traitement de la transaction.
Notons que le code de récupération sera entre les mains du bénéficiaire accompagné de sa pièce d'identité.
par le biai de cette code que il recuperez l'argent aupres de l'Agent.

Moi

Mes informations personnelles

Dans cette page, vous trouverez les informations suivantes :
1. Informations sur votre compte, telles que votre identité utilisateur
2. Formulaire de changement de mot de passe, que vous remplissez de la même façon que lorsque vous oubliez votre mot de passe
3. A propos de nous, la version et la Politique d'utilisation
4. QR code de vos comptes, lorsque vous cliquez sur les numéros de vos comptes CDF et USD, ces codes sont générés automatiquement et affichés en QRCode.
En présentant ces codes QR à un Agent, celui-ci peut automatiquement récupérer le numéro de votre compte et effectuer une transaction (par exemple, un dépôt) en votre nom.
